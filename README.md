# ACNH_Guide_and_Tracker
This is an Animal Crossing: New Horizons tracking app to track certain progress the game doesn't track in certain areas, e.g. villager friendship points, and provide guides for certain areas. It is cross-platform (Windows 11, macOS, and (I think every distro) Linux).


If I say anything silly or the code is not quite so professional, I am a kid who is just starting to learn how to code - this was just a project idea I had which could help my programming skills, although it provides some useful trackers so I thought I would release it!


The app comes as a .exe installer for Windows 11, and a .zip for macOS and Linux.


The source code is completely available for you - do whatever you want with it; I'm just a kid trying to practice my coding, if you want to make this app better for actually using - go ahead, you have my permission!

I will make the .py and the .iss (for the installer on Windows 11) available here but they are also in the app folder.

On macOS and Linux just "ACNH_Guide_and_Tracker\Program_Files\Source_Code" after extraction, and they are both located in that folder.

On Windows 11, head to "C:\Users\your_user\AppData\ACNH_Guide_and_Tracker\Program_Files\Source_Code" (replace "your_user" with your username and make sure in file explorer, "View hidden items" is on (Look at the bar near the top starting with "New"\Click on "View" on that bar\Hover over "Show"\Click "Hidden items")) after installation.

Also, feel free to ask me to make updates - I would be more than happy to help - this is fun! I don't know if you can comment on this repository or something but I will find a way


It is coded in python using the GUI Tkinter.


This app will receive updates, usually new features to track but possibly bug fixtures if there are any (fingers crossed there aren't).
